# Influence of Human Disturbance on Movement and Habitat Selection in Wild Asian Elephants (Elephas maximus) in Nepal
Scripts used to clean, analyse and plot data for a science project/paper for my school's 'sci fest'

## Datasets
- [Occurrence Points of Wild Asian Elephants (Elephas_maximus) in Banke Bardia and Kanchanpur Districts of Nepal](https://datadryad.org/dataset/doi:10.5061/dryad.dncjsxkwh)
- [OpenStreetMap Data for Nepal from Geofabrik from 2021](https://download.geofabrik.de/asia/nepal-210101.osm.pbf)
